Pour cette édition, Émile Filteau présente le développement d'applications web avec React, Redux et Typescript. Vous verrez les bases des technologies React, Redux et Typescript. Vous devrez ensuite construire une application web à partir d'une interface de programmation donnée. Cet atelier nécessite d'avoir [Yarn](https://yarnpkg.com/lang/en/) installé sur son ordinateur.

## Outils
- Yarn
